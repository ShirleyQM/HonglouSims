# 林黛玉视角文字关卡包

素材来源：

- 第三回：林黛玉进贾府
- 第十七回：大观园题额后，荷包与香袋误会
- 第二十九回：清虚观后，金麒麟、白认得你、砸玉剪穗
- 切分素材：`hongloumeng_original/selected_chapters_conversations/`

定位：原汁原味、有限自由的文字关卡。玩家扮演林黛玉，选择的是“应对策略”，不是现代口吻的自由改写。结局允许小幅偏移：能让误会轻些、让关系更稳，或让黛玉更早学会借紫鹃/贾母/礼法保护自己，但不改动红楼核心人物性情。

## 运行口径

这是素材层关卡，不直接接入当前 JS 运行时。若接入游戏，需要新增一个 `conversation_scene` / `choice_effect` 解释器。

已有系统可对应：

- 性格：黛玉 `mingan / duochou / gupi / gengzhi / xijing / shaochi`，专属性格 `solitude_poetry / wind_tear / frail`
- 技能：小型模拟里黛玉 `礼法 90 / 文才 100 / 洞察 95 / 口才 80 / 微妙 70 / 通变 65`
- 关系轴：好感 `affection`、信任 `trust`、友谊 `friendship`、服从/体恤 `submission/care`
- 管理指标：法度 `order`、人心 `morale`、体面 `dignity`、情分 `affinity`

新增素材层状态建议：

| ID | 名称 | 用途 |
| --- | --- | --- |
| `daiyu_composure` | 持礼 | 黛玉在礼法场景中是否稳住体面 |

| `daiyu_hidden_affection` | 藏情 | 对宝玉不明说的在意 |

## 关卡结构

### 关卡一：步步留心

对应第三回。

玩家目标：在荣国府第一次亮相，完成拜见、识人、安顿。

核心体验：

- 看见门第和仆从规格，理解“此处与别家不同”。
- 在贾母痛哭、凤姐热络、宝玉初见之间，保持黛玉式克制。
- 可稍微改变结局：若处理得体，黛玉不必夜里独自过度自责，袭人的劝慰会变成“初步信任”；若过分冷硬或多言，`daiyu_wounded_pride` 增加，后续更容易触发反话试探。

关键选项解锁：

- `礼法 >= 80`：能准确选座、称呼、拜见顺序。
- `洞察 >= 80` 或 `mingan`：能看出凤姐热络背后的控场。
- `诗才/文才 >= 80`：宝玉取字时可含蓄接话，不必只退让。
- `gupi` 或 `xijing` 高：可选择少言观察，但会降低当场亲近。

### 关卡二：荷包香袋

对应第十七回后段。

玩家目标：处理“宝玉身上佩物被小厮解走，黛玉误以为自己的荷包也被给人”的误会。

核心体验：

- 黛玉不是无理取闹，而是在用私物确认“我是否被珍重”。
- 宝玉证明荷包贴身收着后，玩家可以选择更快缓和，或坚持自尊。
- 可稍微改变结局：原书中两人仍和好；游戏可分为“嗤笑缓和”“半日冷战”“紫鹃转圜”三种轻微差异。

关键选项解锁：

- `洞察 >= 85`：看出宝玉贴身藏荷包不是敷衍。
- `微妙 >= 70`：用一句轻讽收住场面，不把争执闹到贾母处。
- `duochou/mingan` 高且 `baoyu_reassurance` 低：更容易剪香袋、追加委屈。
- `poetry >= 2` 或 `solitude_poetry`：可用“香袋未成，心意未尽”的诗性说法转入缓和。

### 关卡三：白认得你

对应第二十九回后段。

玩家目标：在金玉压力、宝玉探病、反话试探中，决定误会升级到什么程度。

核心体验：

- 玩家先经历清虚观的金麒麟和张道士提亲压力。
- 黛玉的反话来自自尊和试探，不是现代直白吃醋。
- 宝玉说“白认得你”后，玩家选择如何回刺、停顿或求证。
- 可稍微改变结局：仍会爆发，但可避免吐药和长辈大规模问责；也可让紫鹃/袭人更早介入。

关键选项解锁：

- `洞察 >= 90`：看出宝玉其实被提亲刺痛。
- `通变 >= 65`：把“听戏去罢”改成委婉留人，降低冲突热度。
- `gengzhi`：可直问“你心里究竟如何”，直接但失红楼含蓄，体面略降。
- `frail` + `daiyu_wounded_pride >= 2`：哭急伤身风险上升。
- `zijuan.trust >= 60`：可让紫鹃提前打断，进入“随侍调停”分支。

## 自由模式事件池

这些事件不是原书逐字剧情，而是从原书互动模式泛化出来：

| 事件 | 触发 | 玩法 |
| --- | --- | --- |
| 信物被误会 | 宝玉/黛玉携带、赠予、遗失私物 | 检查信任、好感、微妙，生成反话试探 |
| 金玉话题 | 宝钗、湘云、张道士、麒麟、玉进入同场 | 黛玉获得冷笑/沉默/离席/转问四类选项 |
| 丫鬟调停 | 黛玉或宝玉负面状态高 | 紫鹃/袭人根据信任与职责介入 |
| 长辈压场 | 争执热度高且贾母/王夫人同场 | 直接终止争执，改为问责随侍或安抚晚辈 |
| 诗性转圜 | 黛玉文才高、场景为潇湘馆/园中 | 用诗文、物件、景物把冲突转成含蓄和解 |
