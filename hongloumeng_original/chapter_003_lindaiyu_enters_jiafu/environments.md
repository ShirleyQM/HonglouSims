# 第三回环境/礼法描写试切

## 宁国府门前
街北两个大石狮子，三间兽头大门，门前列坐十来个华冠丽服之人；正门不开，只东西角门有人出入，匾书“敕造宁国府”。

game use: 展示贾府门第、礼制和权势压迫感。

## 荣国府入门路径
由西角门进入，轿行一箭之远，转弯歇轿，换眉目秀洁的小厮抬至垂花门；婆子扶黛玉下轿。

game use: 体现内外有别、等级分工、黛玉步步留心。

## 贾母正房大院
垂花门内，两边超手游廊，正中穿堂，紫檀架子大理石屏风；转过屏风为三间厅房，厅后正房大院；正面五间上房雕梁画栋，两边游廊厢房，挂鹦鹉画眉等鸟。

game use: 贾母生活核心空间，热闹、华贵、规矩森严。

## 贾赦处院落
从黑油漆大门入，过仪门至院中；正房、厢房、游廊小巧别致，不似贾母处轩峻壮丽，院中树木山石皆好。

game use: 与荣府正院对比，偏独立、精致但权威感稍弱。

## 荣禧堂
大甬路直通大门，仪门内大院五间正房，轩昂壮丽；堂内赤金九龙青地大匾写“荣禧堂”，紫檀雕螭案、青绿古铜鼎、墨龙大画、錾金彝、玻璃盆、楠木圈椅、乌木联牌錾金字。

game use: 贾政/王夫人一脉的正统权力空间，适合做家族威仪背景。

## 王夫人东耳房
临窗大炕铺猩红洋毯，设大红金钱蟒引枕、秋香色金钱蟒条褥、梅花式洋漆小几、文王鼎、汝窑美人觚与时鲜花草；椅搭、脚踏、茗碗瓶花俱备。

game use: 内宅日常议事处，陈设华贵但较生活化。

## 贾母晚饭场景
贾母正面榻上独坐，黛玉按客位左首就坐，三春依序坐定；李纨捧杯，熙凤安箸，王夫人进羹；外间仆妇丫鬟虽多，连咳嗽声也无。

game use: 饭桌礼法、沉默服务、黛玉初学贾府规矩。

## 碧纱厨与暖阁
贾母安排黛玉暂住碧纱厨，宝玉在外面床上；每人奶娘丫头照管，外间上夜听唤，熙凤送来藕合色花帐、锦被缎褥。

game use: 宝黛初入同一生活圈，亲密但仍有礼制隔断。
