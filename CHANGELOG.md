# CHANGELOG

## v0.1.0-alpha · HonglouSims

发布日期：待发布

### 版本主题

第一版公开试玩，重点验证人物 AI、起居、传令、家具互动和社交互动的基础体验。

### 新增

- 起居面板第一版：
  - 48 刻时间网格。
  - 需求、技能、职业作息。
  - 跨日就寝显示。
  - 职业日课和随侍轮值投影。

- 传令面板第一版：
  - 统一“传令”入口。
  - 点名传令和群体传令左右分栏。
  - 当值“令”角标可快捷打开传令。
  - 候选卡展示服从、信任、预计接受、职责内和任务负载。

- Utility AI 接入：
  - 起居锚点影响候选权重。
  - 任务和随侍优先级接入 AI 决策。
  - 旧日程和基础作息作为兜底。

- 后台 v2：
  - 导航按策划视角重组。
  - 人物配置、关系与身份、任务与传令、行为规则、世界配置、AI 与调试分组。
  - 未原生化模块已接入 v2 外壳。

- 测试体系：
  - `tests/README.md`
  - 配置体检。
  - 前台冒烟。
  - 后台冒烟。
  - 传令面板冒烟。
  - 场景测试注册表。

### 修复

- 修复起居跨午夜活动显示和移动规则。
- 修复起居活动全局不重叠规则。
- 修复人物可能被社交/跟随逻辑推到空白地图边缘的风险。
- 修复 favicon 缺失导致的浏览器 404 warning。

### 已知问题

- 群体传令需要具备权限的角色才会出现可选项。
- 多日模拟场景测试还没有完全自动化。
- 本地 LLM 功能不适合直接放到 GitHub Pages 试玩版中。
- 公开试玩版主要依赖浏览器本地存档。
