# 任务完成指南

## 一、任务从哪来、在哪看

| 来源 | 说明 |
|------|------|
| **日常** `daily` | 到点自动下发给指定角色（如袭人 5 点晨起、10 点整理书房） |
| **差遣** `directive` | 上位者 AI 或玩家：选角色 → 点目标 → **差遣** → 选任务 |
| **故事** `event` | 人生路径节点解锁后自动下发（袭人/凤姐线），右下角任务列可见 |
| **群组** `group` | 一次下发给多人（如「小姐们请安」「全院洒扫」） |

**操作**

| 操作 | 方式 |
|------|------|
| 下发任务 | 选角色 → 点目标人物 → 差遣 → 选任务 |
| 接/拒任务 | 左下任务列 → 展开 → 接受/拒绝 |
| 看进度 | 右下角任务面板，条件行显示 `{current}/{target}` |
| 收回差遣 | 任务面板 → 我下发的 → 收回（仅待回应） |

故事任务：切换控制袭人/凤姐时看自己的任务；控制宝玉等人时，面板会显示**其他路径角色的故事任务**（只读提示）。

## 二、完成条件怎么理解（核心）

进度按**游戏分钟**累加（不是现实秒数）。接受任务后，满足条件即 +1 分钟。

| 条件类型 | 你要做什么 |
|----------|------------|
| `USE_FURNITURE_DURATION` | 走到家具旁 → **点击使用** → 保持在使用阶段（`use`），每分钟 +1 |
| `USE_FURNITURE` | 完整使用家具**一次**（用完弹出结束），计 1 次 |
| `INTERACT_WITH` | 走到目标人物旁 → **互动**（交谈/问安等）完成一整轮交互 |
| `INTERACT_WITH_DURATION` | 与目标处于**交谈或互动进行中**，同场景且交互未断，每分钟 +1 |
| `FOLLOW_CHARACTER` | 与目标**在同一场景**，每分钟 +1（不必贴在一起，但不能分场景） |
| `STAY_IN_SCENE` | 身体在指定场景内，每分钟 +1（可站着不动或做别的事） |

**家具与技能**

- **红木书案**（`desk`）：需要 **诗词** 技能，适合宝玉、黛玉、探春抄书作诗。
- **樟木案几**（`workdesk`，图标 📋）：**无技能要求**，整理/洒扫/缝补/算账任务请用这个。
- **饭桌**（`meal`）、**梳洗妆台**（`wardrobe`）：生活必需品，无额外技能门槛。

> 任务文案写「书案/书桌」时，仆从和凤姐应点 **樟木案几**，不要点红木书案（会提示「需技能诗词」）。

## 七、已知冲突与处理（已修 / 需注意）

### ✅ 已修复：整理书桌 vs 诗词技能

- **问题**：整理书房、洒扫、缝补、查账等任务计 `desk`，但红木书案要诗词，袭人/凤姐/仆从做不了。
- **修复**：新增 **樟木案几**（`workdesk`），任务计进度时书案+案几均可；仆从请点 📋 案几。

### ⚠️ 仍需注意

| 情况 | 说明 |
|------|------|
| 1001 罚抄 | 仍须**红木书案 + 诗词**；无诗词角色做不了（设计如此） |
| 4002 夜补雀金裘 | AI 下发要求女红 3 级；路径前期女红不足时任务不会出现，多做女红相关行动 |
| 1002 禁足 | 不能进大观楼（场景 3），否则立刻失败 |
| FOLLOW 类 | 目标人物若一直换场景，需**跟着走**保持同场景 |
| INTERACT_WITH_DURATION | 必须持续交互中；中途走开对话断了就不计分钟 |
| 私宅权限 | 仆从进小姐院子常需跟随主人或任务授权；跟丢宝玉会导致跟随类完不成 |
| 5005 贪墨 | 完整分支选择 UI 未做，目前互动来旺即可推进任务节点 |
| 月钱发放 | 文档写月初；当前为每日 5007 占位，非真实历日 |

## 八、快速排查「任务完不成」

1. 任务是否 **已接受**（非待回应）？
2. 右下角进度 `{current}/{target}` 是否在涨？
3. **家具类**：是否点对了案几（📋）？是否处于使用中？
4. **互动类**：目标是否在同场景？是否点的是任务要求的那个人？
5. **跟随类**：宝玉/黛玉是否换院了？你是否跟过去？
6. **时限**：是否已超时失败？
