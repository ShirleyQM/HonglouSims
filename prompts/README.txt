本目录存放大模型提示词，编辑后刷新游戏即可生效（须通过 HTTP 服务打开，非 file://）。

interaction-global.txt   全局系统提示：世界观、口吻、类别基调、输出格式
interaction-topics.txt   各互动名称的具体侧重（寒暄、打趣、试探等）

程序自动拼装的单次请求上下文包括：
  互动说明：类别基调、具体侧重、模板参考语气
  场景实况：公私属性、时节天气、距离、在场他人
  人物档案：身份、口吻、性格、性情、属性、技能、需求、状态、记忆
            对对方的单向关系感受、任务、后续打算
  双向关系：四轴分值与亲疏解读、服从、旧事注
  分段：每段 4–10 句，层层递进；播放当前段时后台预写下一段
  已有对话：续写时衔接（说话人用简称，勿甲乙）
  互动后状态：叙事气泡提示（尴尬、恼怒等，见 0609_02 文档）

配置项（js/config.js → narrativeBubble.settings.llm）：
  promptGlobalPath   全局提示路径
  promptTopicsPath   话题侧重路径
  systemPrompt       若非空则覆盖 global 文件
