# 人物在场互动模板

## 核心机制

**不是"特定人物触发"，而是"人物标签触发"**。

每个在场人物拥有若干标签，不同标签对同一互动有不同的反应。标签可叠加，优先级按列表从上到下，触发第一个匹配的。

### 标签库（示例）

| 标签 | 行为倾向 | 典型人物（示例） |
|------|----------|------------------|
| 风雅 | 真心欣赏文艺，会驻足倾听、品评 | 黛玉、宝玉、妙玉、探春 |
| 附庸风雅 | 假装懂，发表好笑/不靠谱的评论 | 薛蟠、贾环、赵姨娘 |
| 贪嘴 | 看到吃的就走不动道 | 湘云、刘姥姥、宝玉 |
| 挑剔 | 看什么都想挑毛病 | 黛玉、王夫人、凤姐 |
| 热心 | 喜欢帮忙、关怀别人 | 袭人、平儿、李纨 |
| 好事 | 喜欢凑热闹、八卦、传话 | 小红、麝月、凤姐 |
| 刻薄 | 嘴上不饶人，挑刺或讽刺 | 黛玉、凤姐、晴雯 |
| 沉闷 | 不爱说话，静静旁观 | 迎春、惜春、李纨 |
| 幽默 | 喜欢开玩笑、逗乐 | 湘云、刘姥姥、宝玉 |
| 稳重 | 端庄持重，不轻浮 | 宝钗、王夫人、李纨 |
| 怯懦 | 胆小怕事，容易惊慌 | 尤二姐、香菱 |
| 精明 | 眼明心亮，看得透 | 凤姐、探春、宝钗 |
| 痴情 | 情感丰沛，易被情事打动 | 宝玉、黛玉、龄官 |
| 莽撞 | 做事冲动，不计后果 | 晴雯、芳官、薛蟠 |

# 第三部分：机制使用说明

## 触发流程

```
玩家选择家具 → 选择互动动作 → 系统检测在场人物标签
    ↓
按标签优先级匹配第一个
    ↓
触发对应文案 + 属性变化
    ↓
若触发负面效果，可能连锁触发新状态
```


## 多人场景

若在场有**多个带标签的人物**，则分别触发各自的反应，文案拼接呈现：

> 例子：你在抚琴，黛玉（风雅）驻足倾听，薛蟠（附庸风雅）也在场。
>
> 显示：
> "黛玉凝神听着，微微点头。"
> "薛蟠在一旁跟着摇头晃脑，其实根本不在拍子上。"

各自属性变化独立结算。
